초등 수학 단원 위계 거미줄 1~6학년

학기·단원 격자에 배치된 71개 단원을 110개의 선수–후속 관계로 연결한 거미줄. 단원을 클릭하면 거슬러 올라가는 학습 결손 경로와 이어지는 학습 갈래가 모두 강조됩니다.

연결 표시:
읽는 법

• 가로축 = 학년·학기 (1-1부터 6-2까지 12개 학기) · 세로축 = 단원 번호 (1~6)

• 단원 카드 색 = 5대 영역 (수와 연산 / 도형 / 측정 / 변화와 관계 / 자료와 가능성)

• 화살표 색 = 출발 단원의 영역 · 화살표 방향 = 선수→후속 학습 흐름

• 단원 클릭 시: 모든 선수 학습(빨간 강조)과 후속 학습(초록 강조)이 표시되며, 우측 패널에서 학습 내용을 확인할 수 있습니다.
